# Инструкция по работе с Markdown
## Выделение текста

Чтобы выделить текст курсивом необхадимо обрамить его звездочками (*). Например, *вот так*.

Чтобы выделить текст жирным необхадимо обрамить его 2мя звездочками (**). Например, **вот так**.
